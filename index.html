HTML Basis

HTML (HyperText Markup Language) is de standaardtaal voor het maken van webpagina's. Met HTML geef je structuur aan je webpagina door verschillende elementen te gebruiken.

Basis HTML-structuur

Een HTML-document begint altijd met een doctype declaratie, gevolgd door een html-element dat een head- en body-element bevat.

Uitleg van de elementen:

  • <!DOCTYPE html>: Vertelt de browser dat dit een HTML5-document is
  • <html>: Het hoofdelement van een HTML-document
  • <head>: Bevat meta-informatie over het document (zoals de titel en links naar CSS-bestanden)
  • <title>: De titel van de webpagina die in het browsertabblad wordt weergegeven
  • <body>: Bevat de zichtbare inhoud van de webpagina

CSS Basis

CSS (Cascading Style Sheets) is de taal die we gebruiken om HTML-elementen op te maken. Met CSS bepalen we hoe onze webpagina eruit ziet.

Hoe werkt CSS?

CSS werkt door selectoren te gebruiken om HTML-elementen te selecteren, en vervolgens eigenschappen aan deze elementen toe te kennen.

Basis CSS-concepten:

  • Selectoren: Gebruikt om HTML-elementen te selecteren (bijvoorbeeld: h1, .class, #id)
  • Eigenschappen: Wat je wilt veranderen (bijvoorbeeld: color, background-color, font-size)
  • Waarden: De waarde die je aan de eigenschap wilt geven (bijvoorbeeld: red, 16px, #FFF)
